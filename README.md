# booking-seat-system
Для представления зрительного зала используется двумерный массив символов (char[][]), позваляющий наглядно хранить состояние каждого места.

В коде определены константы FREE_SYMBOL ('F') и BOOKED_SYMBOL ('B'), а также стандартные размеры зала DEFAULT_ROWS (ряд) и DEFAULT_SEATS (место). Это делает код удобным для изменения.

Решение построено на использовании статических методов:
* initHall — инициализация зала свободными местами;
* displayHall — отображение текущего состояния зала в виде таблицы;
* bookSeat / cancelBooking — методы бизнес-логики с проверкой корректности ввода (обработка выхода за границы массива) и проверкой текущего статуса места;
* printStatistics — расчёт и вывод актуальных данных о заполняемости зала.

Интерфейс реализован в виде текстового меню на базе цикла while(true) и конструкции switch-case, обеспечивающие интерактивность и непрерывную работу программы до выбора пользователем пункта «Выход».
